# Nexora Ventures | Redefining the Future of Real Estate

Nexora Ventures is a premium, high-converting real estate solutions platform designed for the Pakistani market. It integrates development, consultancy, and marketing into a seamless, high-end digital experience.

![Nexora Banner](/public/hero_bg.png)

## 💎 Premium Design Philosophy
Nexora is built with a "Deep Dark Chocolate & Walnut" aesthetic, utilizing a custom 7-token design system to deliver an untouchable, native-app feel.

- **Dynamic Personalization**: Cookie-based user recognition for a tailored advisor-style experience.
*   **Cinematic Transitions**: 0.4s split-screen wipe transitions between all pages.
- **Fluid Motion**: Powered by GSAP, Lenis Smooth Scroll, and custom Intersection Observers for 10+ premium animation systems.
*   **Architectural Grid**: A disciplined, responsive layout reflecting architectural precision.

## 🚀 Technology Stack
- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS (Tailwind-free for maximum flexibility)
- **Animations**: GSAP (GreenSock), ScrollTrigger
- **Icons**: Lucide React
- **Smooth Scroll**: Lenis
- **User Experience**: Custom Cookie-based Personalization logic

## 🏗️ Core Features
- **Entry Experience**: Personalized gatekeeper for first-time visitors.
- **The Path to Excellence**: An interactive four-phase methodology (Consultation, Strategy, Execution, Handover).
- **Leadership Showcase**: High-impact professional highlight cards for executive team.
- **Integrated Services**: Full-spectrum clarity across all real estate verticals.
- **Signature Projects**: High-fidelity showcases for premium developments like *Premio Homes*.

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/musman2403/nexora.git
   ```
2. Navigate to the project directory:
   ```bash
   cd nexora
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
